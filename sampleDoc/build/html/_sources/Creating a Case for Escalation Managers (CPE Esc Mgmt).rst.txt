Creating a Case for Escalation Managers (CPE Esc Mgmt)
=======================================================

You can create a JIRA ticket of the CPE Esc Mgmt type if you want to adress an issue directly to a CPE Escalation Manager. The CPE Escalation Manager team belongs to the cpe_esc_mgmt team in VictorOps and follows an on-call rotation policy, so at all times there will be one CPE Escalation Manager on call. When creating a CPE Esc Mgmt, if we want for the CPE Escalation Manager to be contacted (following his or her personal paging policy) you must check the 'Notify Esc Mgmt:' box:
